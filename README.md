# RealityValue - 真实的值

一个为 Minecraft 添加健康、理智、口渴系统的 NeoForge 模组。

## 功能介绍

### 健康系统
- 玩家拥有额外的健康值（0-20），独立于 Minecraft 原版生命值
- 健康值低于 6 时：
  - 每 20 秒获得**反胃**效果
  - 发送警告提示
- 健康值高于 10 时，每 5 分钟自动恢复 1 点健康

### 理智系统
- 玩家拥有理智值（0-20）
- 理智值低于 7 时获得**虚弱**效果和随机负面效果
- 理智值低于 4 时，每 10~50 秒在动作栏显示情绪低落消息
- 理智恢复方式：
  - 钓鱼、采花
  - 睡觉（每 90 秒恢复 1~3 点）
  - 靠近光源（每 12 分钟恢复 1~2 点）
- 理智自然衰减：每 6 分钟降低 1 点
- 雨天/水中：每 2 分钟降低 1 点

### 口渴系统
- 联动 [ThirstWasTaken](https://www.curseforge.com/minecraft/mc-mods/thirst-was-taken) 模组
- 玩家拥有口渴值（0-20）
- 朗姆酒可恢复口渴值

## 物品

| 物品 | 稀有度 | 功能 |
|------|--------|------|
| 草药 | 普通 | 破坏草类 15% 掉落，恢复 2 血量和 0~1 健康 |
| 破碎布料 | 普通 | 击杀僵尸掉落，恢复 2~4 生命值 |
| 绷带 | 稀有 | 由草药+破碎布料合成，恢复 4 血量和 1~3 健康 |
| 抗生素 | 珍贵 | 破坏苔藓 1% 掉落，恢复 8 血量和 4~7 健康 |
| 朗姆酒 | 珍贵 | 恢复 6~14 理智和 3~6 口渴值 |
| 急救针 | 极品 | 直接回满生命值，恢复 10 健康 |

## 模组联动

### Create
以下物品使用后恢复 3~4 点理智：
- 巧克力 (Bar of Chocolate)
- 巧克力覆层浆果 (Chocolate Glazed Berries)
- 建筑者茶饮 (Builder's Tea)
- 蜂蜜苹果 (Honeyed Apple)
- 甜甜卷 (Sweet Roll)

### Farmer's Delight
所有菜品（食物和饮品）使用后恢复 3~6 点理智
- 排除：狗粮、马食、作物类（卷心菜、番茄、洋葱、水稻等）

### ThirstWasTaken
- 低纯度水（肮脏、有点脏）饮用后损失健康和理智
- 朗姆酒通过 ThirstWasTaken API 恢复口渴值

## 战斗机制

- 僵尸攻击玩家时：
  - 75% 概率损失 1~3 点健康
  - 30% 概率损失 1~2 点理智
- **举盾时免疫理智损失**
- 其他生物攻击不会影响健康和理智

## HUD 显示

- 物品栏上方显示健康值和理智值图标
- 使用物品时屏幕中显示进度条和物品名称

## 配置要求

- Minecraft 1.21.1
- NeoForge
- Java 21

## 安装

1. 下载最新的 release 版本
2. 将 `.jar` 文件放入 `mods` 文件夹
3. 启动游戏

## 开发

```bash
# 克隆仓库
git clone https://github.com/createmeow/realityvalue.git

# 构建模组
./gradlew build

# 运行客户端
./gradlew runClient
```
