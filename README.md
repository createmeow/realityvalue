# RealityValue - 真实的值

一个为 Minecraft 1.21.1 增加独立**健康值（Wellness）**与**理智值（Sanity）**系统的 NeoForge 模组。

## 功能介绍

### 健康系统

- 玩家拥有独立的健康值（0~20），与原版生命值相互独立。
- 健康值低于 6 时：
  - 每 10 秒获得一次**反胃**效果；
  - 在聊天栏收到健康警告。
- 健康值高于 10 时，每 5 分钟自动恢复 1 点健康。
- 死亡后会重置为满值。

### 理智系统

- 玩家拥有独立的理智值（0~20）。
- 理智值低于 7 时：
  - 每 6 秒获得**虚弱**效果；
  - 随机获得缓慢 / 黑暗 / 失明 / 反胃中的一种负面效果。
- 理智值低于 4 时：
  - 每 10~50 秒在动作栏显示一条情绪低落消息。
- 理智恢复方式：
  - 钓鱼（50% 概率 +1 理智）；
  - 采摘自然生成的花（50% 概率 +1 理智）；
  - 睡觉（每 90 秒恢复 1~3 点理智，睡醒后回满）；
  - 靠近光源（每 12 分钟恢复 1~2 点理智，灵魂火把等不会提供恢复）。
- 理智自然衰减：每 6 分钟无条件降低 1 点理智。
- 环境惩罚：在水中或露天雨天时，每 2 分钟降低 1 点理智。

## 物品

| 物品 | 稀有度 | 获取方式 | 效果 |
|------|--------|----------|------|
| 草药 | 普通 | 破坏草类 15% 掉落 | 恢复 2 点生命值、0~1 点健康 |
| 破碎布料 | 普通 | 击杀僵尸掉落 | 恢复 2~4 点生命值 |
| 绷带 | 稀有 | 草药 + 破碎布料合成 | 恢复 4 点生命值、1~3 点健康 |
| 抗生素 | 稀有 | 破坏苔藓类方块 1% 掉落 | 恢复 8 点生命值、4~7 点健康 |
| 朗姆酒 | 稀有 | 钓鱼 / 宝藏 / 村民交易等 | 恢复 6~14 点理智 |
| 急救针 | 史诗 | 僵尸稀有掉落 / 地牢宝箱 | 回满生命值，恢复 10 点健康 |

## 战斗机制

- 僵尸攻击玩家时：
  - 75% 概率损失 1~3 点健康；
  - 30% 概率损失 1~2 点理智；
  - **举盾可免疫理智损失**。
- 其他生物攻击不会影响健康值和理智值。

## HUD 显示

- 原版生命值左侧显示健康值图标；
- 原版饱食度右侧显示理智值图标；
- 两者均使用 `leftHeight` / `rightHeight`，可被其他模组的 HUD 条自动顶开；
- 使用物品时屏幕中央显示进度条与物品名称。

## 模组联动

### Create

以下物品使用后可恢复 3~4 点理智：

- 巧克力（Bar of Chocolate）
- 巧克力覆层浆果（Chocolate Glazed Berries）
- 建筑者茶饮（Builder's Tea）
- 蜂蜜苹果（Honeyed Apple）
- 甜甜卷（Sweet Roll）

### Farmer's Delight

所有食物与饮品使用后可恢复 3~6 点理智。

以下物品被排除：

- 狗粮、马食；
- 作物及原材料（卷心菜、番茄、洋葱、水稻、种子、野生作物、稻草等）。

### ThirstWasTaken

- 低纯度水（肮脏、有点脏）饮用后会损失健康值与理智值；
- 肮脏的水还会额外附加饥饿与反胃效果；
- 朗姆酒会通过 ThirstWasTaken API 恢复口渴值；未安装 ThirstWasTaken 时仅恢复理智值。

## 配置要求

- Minecraft 1.21.1
- NeoForge
- Java 21

## 安装

1. 下载最新的 release 版本；
2. 将 `.jar` 文件放入 `mods` 文件夹；
3. 启动游戏。

## 开发

```bash
# 克隆仓库
git clone https://github.com/createmeow/realityvalue.git

# 构建模组
./gradlew build

# 运行客户端
./gradlew runClient
```
